## TechGig CodeGladiators Problems

This repository contains solutions to TechGig CodeGladitors contest. Code Gladiators is an annual coding competition by TechGig, that draws the best and the brightest coding talent from all over India. With multiple contests in emerging technologies and the coveted title of Code Gladiators up for grabs, the competition sees enthusiastic participation and has grown from strength to strength with each passing year. In the initial Open Coding Round, there have been 5038 registrations at the time of this writing.
